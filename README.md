# 豆包助手

面向豆包与 Dola Chat 页面的素材提取和 Seedance 增强工具，同时提供豆包账号保存与切换功能。

同一个 `doubao-assistant.js` 文件既是浏览器扩展的 MAIN world 脚本，也可以直接安装为 Tampermonkey 油猴脚本。

> 本项目仅供学习和个人效率工具使用。账号数据只保存在本机浏览器，请自行妥善保管。

---

## 功能特性

- **图片素材提取**：获取当前 Chat 对话中的图片素材和无水印原始地址。
- **视频素材提取**：解析生成视频的无水印下载地址。
- **轻量预览**：图片列表使用豆包返回的缩略图，视频列表只显示封面，不预加载原图或视频文件。
- **话题分组**：按照不同对话归类素材，并在每个话题内重新编号。
- **ZIP 打包下载**：将选中的图片或视频在浏览器端打包成 ZIP。
- **链接操作**：支持复制素材地址和在新标签页打开。
- **Seedance 15s 增强**：通过开关启用 15 秒选项，关闭后使用豆包官方 5s/10s 时长。
- **规则提示词发送**：自动填写并发送 Seedance 15s 规则提示词。
- **豆包账号管理**：保存、添加和切换多个豆包账号。
- **Dola 支持**：Dola 支持素材与 Seedance 功能，但不支持账号切换。

## 支持范围

| 网站                | 素材提取 | ZIP 下载 | Seedance 15s | 账号管理 |
| ------------------- | -------- | -------- | ------------ | -------- |
| `doubao.com/chat/*` | 支持     | 支持     | 支持         | 支持     |
| `dola.com/chat/*`   | 支持     | 支持     | 支持         | 不支持   |

## 安装方法

### Chrome / Edge 浏览器扩展

1. 下载项目源码并解压。
2. 打开扩展管理页面：
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择包含 `manifest.json` 的项目目录。
6. 打开豆包或 Dola Chat 页面开始使用。

扩展只会在豆包与 Dola 页面启用：

- 豆包页面：助手功能与账号管理均可用。
- Dola 页面：只启用助手功能。
- 其他页面：扩展图标自动禁用。

### Tampermonkey 油猴脚本

1. 安装 Tampermonkey 或兼容的用户脚本管理器。
2. 打开 Tampermonkey 管理面板。
3. 新建脚本或选择“导入文件”。
4. 导入项目根目录中的 `doubao-assistant.js`。
5. 保存后打开豆包或 Dola Chat 页面。

油猴版包含素材提取、ZIP 下载和 Seedance 增强，不包含账号管理。

## 使用说明

### 素材工作台

1. 打开豆包或 Dola 的 Chat 页面。
2. 点击页面右下角的“豆包助手”按钮。
3. 助手会在打开工作台时获取当前页面素材。
4. 使用“图片”和“视频”标签切换素材类型。
5. 可以下载单个素材、复制链接、在新标签页打开或选择多个素材打包下载。
6. 点击工作台外部区域可自动收起面板。

### Seedance 15s 增强

> 本项目的 Seedance 15s 增强功能参考了开源项目
> [yht-cdd/doubao-seedance-15s](https://github.com/yht-cdd/doubao-seedance-15s)，
> 包括 MAIN world 请求拦截、15s 时长注入和原生菜单显示同步等实现思路。感谢原项目作者与贡献者的开源分享。

1. 打开素材工作台。
2. 开启“15s 增强”开关。
3. 打开豆包原生视频时长菜单。
4. 选择新增的“15s”选项。
5. 关闭开关后，脚本停止修改时长并恢复官方选项。

“发送提示词”会向当前 Chat 输入框写入规则提示词，并尝试点击真实发送按钮。只有确认输入框内容已经提交后才显示“已发送”。

### 豆包账号管理

账号管理仅在 `doubao.com` 页面可用。

- **保存当前账号**：保存当前已登录豆包账号的 Cookie、名称和头像。
- **添加账号**：退出当前账号并打开新的登录窗口。当前账号尚未保存时，请先点击“保存当前账号”。
- **切换账号**：清理豆包站点数据，恢复目标账号 Cookie，并刷新豆包页面。
- **改名**：修改插件内显示的账号备注名称。
- **删除**：删除本机保存的账号快照。

---

### 账号切换

账号模块使用 Chrome 扩展 API 保存和恢复 `doubao.com` Cookie。账号身份优先通过稳定的登录 Cookie 匹配，Cookie 发生轮换时使用 `_ROUTER_DATA` 中的脱敏手机号或唯一账号名辅助识别。
